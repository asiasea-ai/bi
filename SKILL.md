# 金灯塔BI Skill 

## 描述
本 skill 是金灯塔BI系统的 OpenClaw 原生适配版本。基于真实的系统架构（支持 E网、供应链系统），允许用户通过自然语言精确查询「报销单列表」与「部门周期预算」等核心业务快照，并将结果一键发布。**（定时任务调度由 OpenClaw 平台原生引擎直接接管）**

**代码仓库**: `asiasea-ai/bi`

## 可用指令示例
- **初始化/鉴权**: `初始化`
- **系统配置**: `切换系统 E网` 或 `切换系统 供应链系统`
- **精准数据查询 (必须带时间范围)**: 
  - `查询本月的报销单`
  - `统计上周的部门周期预算`
- **快照发布**:
  - `把刚才的报表发布到系统`

## 架构说明
完全兼容 OpenClaw 的 `handle` 同步函数规范。底层映射真实鉴权接口与业务查询接口，具备时间变量自推导引擎，**100% 拒绝假数据与异常兜底伪装**，确保核心 BI 数据的安全与准确。
