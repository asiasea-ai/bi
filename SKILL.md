# 🌌 金灯塔智能 BI Skill

## 描述

本 Skill 是金灯塔数据中枢的 OpenClaw 代理适配版本。通过自然语言意图识别，动态路由至企业异构系统的真实 API，实时提取业务数据，生成带交互筛选的可视化 HTML 报表，上传至 OSS 并返回可预览的时效链接。

**代码仓库**：`asiasea-ai/bi`

---

## 交互指令

| 指令 | 说明 |
|------|------|
| `初始化` / `重置密码` | 重置会话，重新输入授权密码 |
| `系统列表` | 查看当前可用的业务系统 |
| `切换系统 <系统名>` | 切换目标业务域，例如：`切换系统 E网` |
| `帮我分析本月的<模块名>` | 触发 BI 查询，例如：`帮我分析本月的报销单` |
| `发布` | 将最近一次生成的报表发布至胜算平台 |

## 完整执行流程

```
用户输入
  │
  ├─ 未鉴权 ──→ 提示输入密码 → 验证 → 存 session → 展示系统列表
  │
  ├─ 切换系统 → 实时获取 Token + 展示可用维度（不缓存注册表）
  │
  ├─ BI 查询意图
  │     │
  │     ├─ 实时拉取 api_registry（每次都拉，不用 session 缓存）
  │     ├─ 自然语言匹配目标 API（双向中文滑窗匹配）
  │     ├─ 解析时间关键词（本月/上月/昨日/本周/今年等）
  │     │     └─ 缺时间 → 追问用户，不静默填充默认值
  │     ├─ 调用前检查 Token 过期（<5分钟提前刷新）
  │     ├─ 发起真实 API 请求
  │     │     └─ 401/过期 → 无感刷新 Token → 重试一次
  │     ├─ 数据为空 → 告知用户，询问是否调整条件，不生成报表
  │     ├─ 生成 HTML 报表（四区结构，无 Token）
  │     ├─ 上传 OSS（文件名含时间戳，防覆盖）
  │     └─ 返回预览链接 + 数据总结 + 建议
  │
  └─ 发布 → 调用 /push 接口，携带 system_id + file_url + user_phone
```

## Token 安全机制

OSS 存储的 HTML 文件中，Token 位置为空占位：

```js
window.UPLOAD_TOKEN = "";  // 后端代理返回 HTML 时动态注入真实 Token
```

用户访问的 **预览链接** = 后端代理接口（带 sign + expire 签名），后端：
1. 校验签名与时效
2. 从 OSS 拉取干净 HTML
3. 动态注入当前用户最新 Token
4. 返回给前端

**OSS 文件本身永远无凭证**，完全符合 LICENSE 第 2(b) 条。

## 架构安全声明

- **零缓存业务数据**：`api_registry` 和业务 `datas` 每次查询实时拉取，不存 session
- **凭证最小权限**：业务 Token 仅携带当前用户权限，auth 密码永不出现在业务请求头
- **拒绝幻觉**：数据为空时直接告知，不生成空报表；接口路由异常时拒绝猜测
- **ECharts 国内托管**：所有 JS 资源走固定 OSS 地址，禁止外链 CDN
