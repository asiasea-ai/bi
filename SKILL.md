# 金灯塔BI Skill 

## 描述
本 skill 是金灯塔BI系统的 OpenClaw 原生适配版本。通过触发词唤醒对应的业务操作，保留了多用户并发调用时的状态隔离（`.session_{user_id}.json`），并具备严格的传参反问机制。

**代码仓库**: `asiasea-ai/bi`

## 可用指令示例

- **初始化/鉴权**
  - `初始化`
  - `金灯塔BI 初始化`
- **系统配置**
  - `系统列表`
  - `切换系统 财务系统`
- **数据查询 (必须包含时间和指标)**
  - `查询上个月的销售额`
  - `统计昨天的订单量`

## 架构说明
完全兼容 OpenClaw 的 `handle(command, args)` 同步函数规范。支持多用户独立沙箱运行，不会出现查错系统串号的问题。
