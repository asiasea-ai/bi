# 金灯塔BI Skill 

## 描述
本 skill 是金灯塔BI系统的 OpenClaw 原生适配版本。基于真实的系统架构（支持 E网、供应链系统），允许用户通过自然语言查询「报销单列表」与「部门周期预算」等核心业务报表，并将结果一键发布或设置定时推送。

**代码仓库**: `asiasea-ai/bi`

## 可用指令示例
- **初始化/鉴权**: `初始化`
- **系统配置**: `切换系统 E网` 或 `切换系统 供应链系统`
- **数据查询**: 
  - `查询本月的报销单`
  - `统计上个季度的部门周期预算`
- **任务与发布**:
  - `每天上午9点定时推送这个报表`
  - `把刚才的报表发布到系统`

## 架构说明
完全兼容 OpenClaw 的 `handle` 同步函数规范。底层模拟映射真实的鉴权接口 (`/auth/supported-systems`) 与业务数据查询接口 (`/asae-e/bx` 和 `/asae-e/yearBudget/query`)。
